# Forging System Mod

A comprehensive item forging system for Divinity: Original Sin 2 that implements a probability-based rarity inheritance system.

## Overview

This mod adds a sophisticated forging system that allows players to combine items with a balanced probability distribution system. The system prevents exploitation while maintaining the excitement of crafting through two main mechanisms:

- **Gravity Well (Different Tiers)**: When combining items of different tiers, results are pulled toward the average tier
- **Tier Break (Same Tiers)**: When combining items of the same tier, there's a chance to ascend to the next tier

## Installation

### For End Users

1. Download the mod from [GitHub Releases](https://github.com/YOUR_USERNAME/dos2-forging-system/releases) or Steam Workshop
2. Place in your DOS2 mods directory:
   ```
   Data/Mods/forging_system_d581d214-2dd4-4690-bb44-e371432f1bfc/
   ```
3. Enable the mod in the game's mod manager

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/dos2-forging-system.git
   ```

## Development

### Prerequisites

- Divinity: Original Sin 2 - Definitive Edition

### Building

Binary files (`.data`, `.bin`, `.lsb`) are generated by the Divinity Engine and are excluded from version control. To build:

1. Open the mod in Divinity Engine 2
2. The editor will automatically generate necessary binary files
